## Steps for Uploading project:

### Publish in npm

1. Make the changes...
2. Copy the styles folder and paste in the dist folder
3. Increment the version
4. run `npm run build`
5. run `npm pack`
6. run `npm publish`

### Then to publish in yarn:

1. Increment the version
2. run `npm pack`
3. run `npm publish`
